# Provenance Policy

This repository uses clean-room templates and synthetic data.

Allowed:

- original plotting code written for this repository;
- generated synthetic data with fixed seeds;
- public documentation links;
- user-provided ideas rewritten from scratch without copying files.

Not allowed:

- private datasets;
- local absolute paths;
- copied paper figures or screenshots;
- unclear third-party snippets;
- binary plotting project files;
- author or license markers removed to make copied material look original.

The checks in this repository are guardrails. They do not prove that every
outside source is safe to publish.
