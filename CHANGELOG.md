# Changelog

## Unreleased

  deterministic synthetic data and Matplotlib native inset axes.
- Added `residual_convergence` for semilog iterative residual plots with
  deterministic synthetic solver traces and stopping-rule guidance.
- Added `spectral_density` for frequency-content plots with deterministic
  synthetic signal data and chart-selection guidance.
- Added `category_small_multiples` for comparing the same category set across
  several panels with a shared y-axis scale.
- Added `paired_before_after` for paired two-condition change plots with
  synthetic gallery output and chart-selection guidance.
- Added `lollipop_ranking` as a lightweight sorted comparison template with
  synthetic gallery output and README/chart-selection coverage.
- Initial v0.1 repository scaffold with 12 Matplotlib templates, gallery
  rendering, provenance docs, bilingual README files, and a quality workflow.
- Added README quality badges, a four-image gallery preview, and public
